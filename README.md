# Simple To-Do List

A clean, modern, and responsive To-Do List application built with HTML5, CSS3, and JavaScript.

## Features

- **Add Tasks**: Quickly add new items to your list.
- **Complete Tasks**: Toggle tasks between active and completed states.
- **Delete Tasks**: Remove tasks you no longer need.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Premium UI**: Smooth animations and a contemporary design aesthetic.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern styling with custom properties, transitions and animations.
- **JavaScript**: Core application logic and state management.
